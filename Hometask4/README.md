# Overview
Create script to get PR(Pull Request) statistics from GitHub. By Andrei Shvedau

## Usage
``` 
./github.py [options] <user> [<repo>] 
./github.py --version 
./github.py (-h | --help)
```

### Defaults
Username -- requiered.

Repository -- https://github.com/octocat/Hello-World
